## 📚 Tentang Proyek

**CampuStore** adalah platform e-commerce khusus untuk mahasiswa yang memungkinkan jual beli produk di lingkungan kampus! 🎒✨

### ✨ Fitur Utama

```
🛒 Halaman Katalog Produk
📦 Sistem Harga & Stok
⭐ Review Produk
🔐 Autentikasi User
📱 Responsive Design
```

## 🚀 Fitur Aplikasi

### 🏪 Halaman Katalog Produk

Pada halaman ini, user dapat:
- ✅ Melihat daftar produk yang tersedia
- ✅ **HANYA** melihat & review produk (tidak bisa beli)
- ⭕ **TIDAK** ada tombol "beli" 
- ⭐ **ADA** fitur review produk

### 💰 Informasi Produk
- 💵 Produk/barang diberi **harga**
- 📊 Produk/barang diberi **stok**
